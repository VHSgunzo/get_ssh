## Dependencies:
```
Chrome or Chromium
random_username
selenium
undetected_chromedriver
```
## To get started:
* **Download the latest revision**
```
git clone https://github.com/VHSgunzo/get_ssh.git && cd get_ssh
```
* **Install python dependencies for get_ssh**
```
pip install -r requirements.txt 
```
